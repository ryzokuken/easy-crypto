Methods

(inner) hash(algorithm, message, inputEncoding, outputEncoding) → {string|Buffer}

Hash a given message using the specified hash algorithm.
Parameters:
Name Type Description
algorithm string The hashing algorithm to be used.
message string | Buffer | TypedArray | DataView The message to be hashed.
inputEncoding string The encoding of the `message`. If nothing is provided and `message` is a string, an encoding of `'utf8'` is enforced. If `message` is a Buffer, TypedArray, or DataView, then inputEncoding is ignored.
outputEncoding string The encoding of the output. If encoding is provided a string will be returned; otherwise a Buffer is returned.
