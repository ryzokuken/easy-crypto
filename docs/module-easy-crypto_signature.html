Module: easy-crypto/signature

The signature module contains a small set of basic utilities for generating and verifying signatures.

(inner) sign(privateKey, algorithm, message, inputEncoding, outputEncoding) → {string|Buffer}

Generate signatures for a message using a private key.
Parameters:
Name Type Description
privateKey Object | string | Buffer | KeyObject The private key to be used for signing the message.
algorithm string The hashing algorithm to be used.
message string | Buffer | TypedArray | DataView The message to be signed.
inputEncoding string The encoding of the `message`. If `message` is a string and no value is provided, an encoding of `'utf8'` will be enforced. Ignored if message is a `Buffer`, `TypedArray` or `DataView`.
outputEncoding string The encoding of the output signature. If provided a `string` is returned, otherwise a `Buffer` is returned.

(inner) verify(publicKey, algorithm, message, signature, inputEncoding, signatureEncoding) → {boolean}

Verify if a signature is valid for a given message using the corresponding public key.
Parameters:
Name Type Description
publicKey Object | string | Buffer | KeyObject The public key to be used for verifying the signature.
algorithm string The hashing algorithm to be used.
message string | Buffer | TypedArray | DataView The message for which the signature has been generated.
signature string | Buffer The signature to be verified.
inputEncoding string The encoding of the `message`. If `message` is a string and no value is provided, an encoding of `'utf8'` will be enforced. Ignored if message is a `Buffer`, `TypedArray` or `DataView`.
signatureEncoding string The encoding of the provided `signature`. If a signatureEncoding is specified, the signature is expected to be a string; otherwise signature is expected to be a Buffer, TypedArray, or DataView.
